# 🚀 Asteroids Clone (Unity 2D)

A simple arcade-style **Asteroids-inspired game** built in Unity, focused on physics-based movement, clean code structure, and gameplay feel.

---

## 🎮 Features

* Smooth **ship movement with rotation and thrust**
* **Projectile shooting system**
* Asteroids that **split once into smaller pieces**
* **Dynamic spawning** from screen edges toward the player
* Score-based survival system:

  * Player takes increasing penalties (**100 → 200 → 400 → 800…**)
  * Game ends when score reaches zero
* **Collision handling** with temporary invincibility
* Basic **juice effects**:

  * Sound effects (shoot, hit, destroy)
  * Explosion particles
  * Screen shake
  * Hit feedback

---

## 🧠 What This Project Demonstrates

* Unity fundamentals (**2D physics, input, prefabs, collisions**)
* Clean component-based architecture
* Game loop design (spawn → interact → score → difficulty)
* Basic game feel improvements (audio + visual feedback)
* Debugging and system iteration

---

## 🎯 Controls

* **A / D** → Rotate
* **W** → Thrust forward
* **S** → Reverse thrust
* **Space** → Shoot

---

## 🛠️ Tech Stack

* Unity (2D)
* C#
* TextMeshPro (UI)
